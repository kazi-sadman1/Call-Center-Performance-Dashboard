## üîç Key Features

### Overall Call Performance Overview
Tracks total calls, answered vs. abandoned calls, success rate, and call outcomes.

### Agent Performance Analysis
Evaluate individual agent metrics including handled calls, success rate, abandoned calls, and productivity.

### Interactive Filtering & Drill-Down
Use slicers to select specific agents and instantly view their performance insights, products discussed, and customer locations.

### Monthly Call Trends
Visualize month-wise call volume and success trends to identify patterns and performance changes over time.

### Unsuccessful Call Insights
Identify and analyze key reasons behind failed or unsuccessful calls to support process improvement.

### Customer Behavior & Location Analysis
Understand customer distribution and preferences based on call data.
